# Akasha Wiki

Personal wiki for exploring the intersection of **Ayahuasca**, **psychotherapy**, and **spiritual healing**. Built following the [Karpathy LLM Wiki pattern](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f).

## The Vision

This wiki supports my journey to help people through the sacred medicine of Ayahuasca combined with psychotherapy. The goal is to build a comprehensive knowledge base for:

- **Research**: Compiling scientific studies on psychedelic-assisted therapy
- **Integration**: Creating frameworks for healing and personal growth
- **Practice**: Developing approaches for ceremonial work and therapeutic integration

## Structure

```
akasha-wiki/
├── raw/                    # Immutable source documents
│   └── ayahuasca-*.md      # Research compilations
├── wiki/                   # LLM-generated content
│   ├── sources/            # Processed source summaries
│   ├── concepts/           # Topic pages
│   └── entities/           # Entity pages (people, studies, institutions)
├── index.md                # Catalog of all wiki pages
├── log.md                  # Chronological activity log
└── CLAUDE.md               # Schema and conventions for LLM agents
```

## Core Concepts

- **Ayahuasca**: Sacred plant medicine from the Amazon
- **Psychedelic-Assisted Therapy**: Clinical use of psychedelics for healing
- **Integration**: Processing and applying insights from ceremonies
- **Ego Dissolution**: The dissolution of self-boundaries during psychedelic experiences
- **Set & Setting**: Mindset and environment for safe ceremonial work

## The Human-LLM Partnership

I curate sources and ask questions. The LLM maintains the wiki — summarizing, cross-referencing, and keeping everything consistent. Together we build a knowledge base that compounds over time.
