╠═══════════════════════════════════════════════════════════════════════════════╣

```

## 2. Farmacologia: Por Que Microdoses Funcionam

### 2.1 O Sistema Serotoninérgico e o Receptor 5-HT2A

```

│                    FARMACOLOGIA DO DMT E 5-HT2A                             │
├─────────────────────────────────────────────────────────────────────────────┤

│   │          │     │                                      │           │   │
│   │          │     │   DOSE ALTA → Gq → PLC → IP3/DAG   │           │   │

│   │              │  → Sinalização via Gq apenas        │              │   │
│   │              │  → Neuroplasticidade sustentada   │              │   │

│   │   • Ativação sutil do Sigma-1                                       │ │

║  │     • Redução de inflamação → ambiente neural mais saudável            │  ║
║  │     • Normalização do eixo HPA → regulação de estresse melhor         │  ║
║  │                                                                          │  ║
║  │  4. A "JANELA DE PLASTICIDADE" EXPLAICA A DURAÇÃO                        │  ║
║  │     • Após microdose, o cérebro fica "mais receptivo" por dias         │  ║
║  │     • Isso permite que outras intervenções (terapia, insights)          │  ║
║  │       sejam mais eficazes                                                │  ║
║  │     • A plasticidade não vem da substância, mas do momento que          │  ║
║  │       a substância cria                                                  │  ║
║  │                                                                          │  ║
║  │  5. O TEMPO CRIA ACÚMULO                                                 │  ║
║  │     • Efeitos de BDNF e inflamação persistem além da substância         │  ║
║  │     • Com doses repetidas, a "janela" se sobrepõe                       │  ║
║  │     • O resultado é neuroplasticidade sustentada                       │  ║
║  │                                                                          │  ║
║  └─────────────────────────────────────────────────────────────────────────┘  ║
║                                                                               ║
║  ─────────────────────────────────────────────────────────────────────────    ║

║  ─────────────────────────────────                                            ║
