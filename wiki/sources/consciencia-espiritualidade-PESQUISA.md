## 1. Teorias Principais da Consciência

### 1.1 Teoria da Consciência Global (Global Workspace Theory - IIT)

**Teórico:** Stanislas Dehaene, Bernard Baars, Jacobo Grinberg-Zylberbaum

A **Teoria da Consciência Global** (Global Workspace Theory - IIT *não confunda com Integrated Information Theory de Tononi*) propõe que a consciência emerge da **dinâmica de competição e Broadcasting** entre processos cerebrais.

```
┌─────────────────────────────────────────────────────┐
│              GLOBAL WORKSPACE                       │
│   ┌──────────────────────────────────────────┐     │
│   │  Broadcasting para todo o cérebro        │     │
│   │  (acesso consciente)                     │     │
│   └──────────────────────────────────────────┘     │
│                        ↑                            │
│         ┌──────────────┴──────────────┐              │
│         │     Competição por          │              │
│         │     acesso ao Workspace     │              │
│         └──────────────┬──────────────┘              │
│    ┌────────┐   ┌────────┐   ┌────────┐              │
│    │ Módulos│   │ Módulos│   │ Módulos│              │
│    │Process.│   │Process.│   │Process.│              │
│    └────────┘   └────────┘   └────────┘              │
└─────────────────────────────────────────────────────┘
```

**Princípios Fundamentais:**
1. **Informação globalmente disponível** — A consciência existe quando informação é "broadcast" para todo o cérebro
2. **Competição neural** — Múltiplos processos competem pelo acesso ao workspace
3. **Automatização inibe acesso** — Processos automáticos (habituais) não accedem ao workspace
4. **Focalização/atenção** — A atenção dirige competição, não é seu resultado

**Evidências Empíricas:**
- Estudos de neuroimagem mostram "picos" de actividade global durante estados conscientes
- O modelo explica por que stimuli subliminares não são conscientes
- Aplicação a Ayahuasca: a redução da DMN (Default Mode Network) pode ser理解为 como "liberando" o workspace de sua auto-referência habitual, permitindo nova competição e acesso a informação não-ordinária

### 1.2 Teoria da Informação Integrada (Integrated Information Theory - IIT)

**Teórico:** Giulio Tononi, Christof Koch

A **IIT** propõe que a consciência é **informação integrada** — a quantidade de consciência corresponde ao grau de integração de informação de um sistema.

```
                    Φ (phi) - consciência medida
                          │
    ┌─────────────────────┴─────────────────────┐
    │              INFORMAÇÃO                    │
    │                 INTEGRADA                  │
    │                    ↓                        │
    ┌───────────────┐    │    ┌───────────────┐  │
    │   Sistema     │────┴────│   Sistema     │  │
    │  Desintegrado │         │   Integrado   │  │
    │   Φ = 0       │         │   Φ > 0        │  │
    └───────────────┘         └───────────────┘  │
    (processos separados)    (consciousness)     │
```

**Métricas IIT:**
- **Φ (phi)** — "Integrated Information" mede a quantidade de consciência
- Consciousness corresponde a informação integrada além da soma de partes
- A teoria explica por que certos sistemas (cérebros) são conscientes e outros (computadores) aparentemente não são

**Implicações para Ayahuasca:**
- Ayahuasca pode aumentar Φ ao criar novos связей entre áreas cerebrais
- A redução da DMN pode paradoxalmente *aumentar* integração ao desintegrar padrões rígidos
- A experiência de "unidade" pode refletir aumento real em informação integrada

### 1.3 Modelo Penrose-Hameroff (ORCH-OR)

**Teóricos:** Roger Penrose, Stuart Hameroff

O **modelo ORCH-OR** (Orchestrated Objective Reduction) propone que a consciência surge de **efeitos quânticos em microtúbulos celulares**.

```
┌─────────────────────────────────────────────────────┐
│              ORCH-OR MODEL                         │
│                                                     │
│  Microtúbulos ──────────────────────────────────    │
│     ↓                                               │
│  Efeitos Quânticos (superposições)                  │
│     ↓                                               │
│  OBjective Reduction (colapso quântico)             │
│     ↓                                               │
│  Momentos de consciência (conscious events)         │
│     ↓                                               │
│  Experiência subjectiva (feeling)                  │
└─────────────────────────────────────────────────────┘
```

**Mecanismo Proposto:**
1. Processos quânticos ocorrem em **microtúbulos** (estrutura celular)
2. Quando a superposição quântica atinge limiar de energia gravitacional, ocorre **Objective Reduction (OR)**
3. Cada OR é um "momento" de experiência consciente
4. A consciência não é contínua — é uma série de "eventos" discretos

**Conexão com Ayahuasca:**
- Ayahuasca atua em receptores 5-HT2A, não diretamente em microtúbulos
- Mas estados de consciência expandida podem ser理解为 como intensificando processos quânticos?
- A "profundidade" da experiência ayahuasca pode envolver maior número ou intensidade de eventos OR

### 1.5 Teoria da Biologia da Consciência

**Teórico:** Antonio Damasio

Damasio propõe que a consciência é fundamentalmente **corporificada** — a mente surge da **interação entre cérebro, corpo e ambiente**.

```
    Corpo (visceral, somático)
          ↓
    Representações somáticas no cérebro
          ↓
    Core Consciousness (sentimento primitivo)
          ↓
    Extended Consciousness (autoconsciência)
```

**Conceitos-Chave:**
- **Core Consciousness** — Sentimento presente, momento a momento
- **Extended Consciousness** — Autobiografia, antecipação do futuro
- **Soma** — O corpo como fundamento da mente
- **Sentiments** — Patterns de respostas corporais

**Ayahuasca e a Teoria de Damasio:**
- Ayahuasca intensifica **interocepção** (percepção do corpo interno)
- Muitos relatam "sentir o corpo de forma diferente"
- A experiência de "unidade" pode envolver dissolução das fronteiras entre self e corpo

### 1.6 Higher-Order Theories of Consciousness

**Teóricos:** David Rosenthal, Georg Northoff

Estas teorias propõem que consciência requer **representações de ordem superior** — pensar sobre pensamento.

```
┌─────────────────────────────────────────────────────┐
│       HIGHER-ORDER THEORY                           │
│                                                     │
│  Estado Mental de 1ª ordem (ex: ver azul)         │
│          ↓                                          │
│  Representação de 2ª ordem ("estou vendo azul")    │
│          ↓                                          │
│  Consciência (awareness)                           │
└─────────────────────────────────────────────────────┘
```

**Implicação:** Sem capacidade de pensar sobre experiências, não há consciência.

## 2. Consciência como Propriedade Fundamental vs. Emergente

### 2.1 O Debate Fundamental/Emergente

Esta é uma das questões mais profundas da filosofia da mente:

```
┌─────────────────────────────────────────────────────────────┐
│  POSIÇÕES SOBRE A NATUREZA DA CONSCIÊNCIA                    │
│                                                              │
│  FUNDAMENTALISMO                    EMERGENTISMO             │
│       │                                  │                   │
│  "Consciência é tão real              "Consciência emerge    │
│   quanto massa ou carga               de processos físicos   │
│   elétrica"                           complexos"             │
│       │                                  │                   │
│  - Panpsychismo                        - Funcionalismo       │
│  - Propriedade fundamental             - Teoria Global        │
│  - Não-redutível                       - IIT (disputed)       │
│       │                                  │                   │
│  Consciousness is brute               Consciousness is        │
│  - there, not derived                  derived from neural    │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 Argumentos para Consciência como Propriedade Fundamental

**Argumento de Chalmers (O "Problema Difícil"):**
```markdown
Por que existe algo em vez de nada?
Por que existe experiência subjetiva?
Se移除 toda a estrutura física do universo, 
o "felt experience" ainda existiria?
Se sim, consciência é fundamental.
```

**Experimentos Mentais:**
- **Quarto Chinês (Searle):** Manipulação de símbolos não produz compreensão
- **Zumbis Filosofóficos:** É logicamente possível um "zumbi" fisicamente idêntico a um ser consciente, mas sem experiência subjetiva
- **Inverted Spectrum:** Imaginem cores invertidas — a experiência é diferente, o comportamento o mesmo

**Implicação:** A consciência pode não ser redutível a processos físicos.

### 2.3 Argumentos para Consciência Emergente

**Emergência em Sistemas Complexos:**
```
Átomos → Moléculas → Células → Redes Neurais → Mente
  ↓        ↓           ↓          ↓            ↓
(química) (biologia) (vida)  (consciência)
   Sem           ↑         ↑         ↑
 consciência     │         │         │
                │         │         │
          Propriedades    │         │
          emergentes      │         │
                    ↑     │         │
                  Propriedades      │
                  emergentes       │
                            ↑       │
                      Finalmente... │
                      CONSCIÊNCIA!  │
```

**Argumento Funcionalista:**
- Se consciência é função de processamento de informação, pode emergir de qualquer sistema que implemente a função
- Não importa o substrato (cérebro, silicone, etc.)
- Ayahuasca demonstra que químicos podem alterar/modular consciência

### 2.4 Abordagem de Nagel: "What Is It Like to Be...?"

Thomas Nagel argumentou que consciência é fundamentalmente **subjetiva** — há algo que é "ser" um determinado organismo.

```
蝙蝠 (morcego)                               │
    │                                         │
    ├── Sonar como mecanismo                │
    │                                         │
    └── Experiência subjetiva ("what it's like")
           │
           │
           └── A consciência é INTRINSECAMENTE
                subjectiva - não pode ser capturada
                por descrições objectivas
```

**Conclusão de Nagel:**
- Consciência é "view from nowhere" — não pode ser objetivamente capturada
- Podemos tentar entender por analogia, mas nunca experienciar diretamente
- AObjectiva completa da consciência é impossível (parafraseando)

### 2.5 Ayahuasca e o Debate Fundamental/Emergente

Ayahuasca apresenta evidências que complicam o debate:

**Para Fundamentalismo:**
- Experiências de "unidade cósmica" sugerem conexão com algo além do cérebro individual
- Relatos de "acesso a informação" que parece existir independentemente
- NDEs mostram que consciência pode persistir/emergir em contextos não-biológicos

**Para Emergencia:**
- Farmacologia清清楚楚: DMT/harminablock/receptor actions explicam mecanismo
- Alterações cerebrais durante experiência são mensuráveis
- Relação dose-resposta sugere processo físico

**Síntese Provisória:**
Ayahuasca pode ser理解为 como **intensificando ou alterando a interface** entre mente individual e "algo mais" — seja este algo fundamental ou emergente de outros sistemas.

## 3. Panpsychismo: A Consciência como Fenômeno Universal

### 3.1 Definição de Panpsychismo

**Panpsychismo** é a view de que a consciência é uma **propriedade fundamental ou universal** de toda matéria — não apenas de organismos biológicos.

```
┌─────────────────────────────────────────────────────┐
│              PANPSYCHISM SPECTRUM                   │
│                                                     │
│  MICRO           │              MACRO              │
│  Panpsychismo    │    Psycomunismo   │  Hylomorphic │
│  radical         │    (chamada       │  (Aristóteles)│
│                  │    "panprotopsychism")│           │
│  "Tudo é         │    "Complexidade  │  "Forma +     │
│  consciente"     │    produz         │  matéria"     │
│                  │    consciência"   │              │
└─────────────────────────────────────────────────────┘
```

### 3.2 Variantes do Panpsychismo

**Panpsychismo Radical (Philip Goff):**
- Toda partícula subatômica tem algum degree of experience
- Experiência humana é apenas "mais desenvolvida"
- Formas simples de consciência existem em toda parte

**Panpsiquismo Neutral (Galen Strawson):**
- A experiência é tão fundamental quanto massa, carga, etc.
- "Feeling" é básico na natureza

**Hylomorphism (Aristóteles moderno):**
- Forma (psyché) + Matéria = Todo ser
- A alma é a forma do corpo vivo

**Russellian Monism (Galen Strawson, et al.):**
- Física descreve apenas comportamento, não natureza intrinsic
- Consciência pode ser a "natureza intrínseca" que a física mapeia extrinsecamente

### 3.3 O "Problema da Combinação" (Combination Problem)

O maior desafio ao panpsychismo:

```
Se partículas têm experiências simples,
como estas combinam-se para formar
experiências complexas como a humana?

┌────┐  ┌────┐  ┌────┐
│ ◉  │  │ ◉  │  │ ◉  │   Experiências simples
└────┘  └────┘  └────┘
    ↓       ↓       ↓
   ???      ???      ???
    └───────┴───────┘
           ↓
    ┌─────────────┐
    │  ◉ ⊕ ◉ ⊕ ◉ │  Experiência complexa
    │ (consciência│  (como?)
    │   humana)   │
    └─────────────┘
```

**O problema:** Como milhões de "micro-experiências" tornam-se "uma" experiência consciente?

### 3.4 Conexão com Ayahuasca

**Experiências que sugerem panpsychismo:**

1. **Sensação de "consciousness of everything":**
   - Relatos de "sentir a consciência das árvores, pedras, universo"
   - Sugere que ayahuasca expande awareness para além do cérebro individual

2. **"O Velo da Maya" (Illusion of Separation):**
   - Muitos relatam que ayahuasca revela que a separação entre self e mundo é "ilusória"
   - Se correto, 支持a panpsychismo — a consciência é fundamental, apenas filtrada

3. **A Interpretação de "Consciência Universal":**
   - Terence McKenna falava do "Mind of the Universe"
   - DMT como "molécula de acesso" a esta consciência mais ampla

**Conexões especulativas:**

## 5. Campo Akásico: Entre a Espiritualidade e a Física

### 5.1 Origens do Conceito

**Tradição Indiana:**
- **Akasha** (आकाश) — o primeiro elemento, "éter" ou "espaço"
- Na filosofia hindu, um dos cinco elementos básicos (panchamahabhuta)
- Considerado o substrato primordial da existência

**Tradição Teosófica:**
- **Registros Akáshicos** — supostamente "codificados" em um plano não-físico
- Desenvolvido por Helena Blavatsky e HP Blavatsky
- Supostamente acessível a videntes e mestres evoluídos

**Ervin Laszlo (2004):**
- "The Akashic Field: A Guide to the Cosmos"
- Propôs o campo akásico como campo físico/matemático
-.Argumentou que explica fenómenos anómalos

### 5.2 Proposta de Laszlo

```
┌─────────────────────────────────────────────────────┐
│  CAMPO AKÁSICO SEGUNDO LASZLO                        │
│                                                      │
│  Vácuo Quântico (Quantum Vacuum)                   │
│       │                                              │
│       ├── "Mar" de energia/flutuação                │
│       ├── Não é vazio, mas plenum (cheio)           │

└─────────────────────────────────────────────────────┘
```

### 5.3 Paralelos com Física Moderna

**Teoria do Vácuo Quântico:**
- Vácuo quântico ≠ vazio clássico
- Flutuações de energia zero-point
- Pares partícula-antipartícula surgem e desaparecem
- "Energia do ponto zero" do vácuo

**Memória Holográfica do Universo (Bohm):**
- David Bohm propôs o "holomovement"
- Tudo está " plegado" em tudo
- Ordem implícita e explícita
- Informação distribuída holográficamente

**Campos Morfogenéticos (Sheldrake):**
- Rupert Sheldrake propôs campos que armazenam padrões
- Moldan forma sem carregar matéria
- "Resonance" de patrones passados
- Controvérsial, mas influente

### 5.4 Criticas e Limitações

**Do ponto de vista científico:**

1. **Não testável empiricamente:**
   - Sem hipótese operacionalizável
   - Não há forma de medir o campo akásico
   - Nenhum experimento de previsão

2. **Problema da especificação:**
   - Como acessar informação específica?
   - Por que vezes accesses correct, outras não?
   - Sem mecanismo de "acesso"

3. **Problema do intermediário:**
   - Como consciência acessa o campo?
   - É necessária uma "interface"?
   - DMT como "antena"?
