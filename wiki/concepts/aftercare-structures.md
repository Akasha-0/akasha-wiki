# Aftercare Structures

## Resumo

Estruturas de aftercare (cuidados posteriores) sao sistemas organizados de suporte oferecidos apos experiencias com ayahuasca para garantir que os beneficios se mantenham e a integracao ocorra de forma sustentavel. Incluem desde acompanhamento individual ate comunidades de pratica continuas.

## Tipos de Estruturas Aftercare

### 1. Acompanhamento Individual

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   ACOMPANHAMENTO INDIVIDUAL                                               │
│                                                                             │
│   Características:                                                         │
│   • Sessoes regulares com terapeuta ou facilitador                       │
│   • Plano personalizado de integracao                                     │
│   • Check-ins de progresso                                                │
│   • Ajuste de praticas conforme necessidade                              │
│                                                                             │
│   FREQUENCIA RECOMENDADA:                                                 │
│   • Semana 1-2: 2x por semana                                            │
│   • Mes 1-2: 1x por semana                                              │
│   • Mes 3+: 1x por mes ou conforme necessidade                           │
│                                                                             │
│   FUNCIONAL PARA:                                                          │
│   • Processamento profundo                                               │
│   • Trabalho com trauma ou experiencias dificeis                         │
│   • Mudancas complexas de habitos                                        │
│   • Quem esta fazendo trabalho therapeutico intensivo                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 2. Grupos de Aftercare

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   GRUPOS DE AFTERCARE                                                     │
│                                                                             │
│   Características:                                                         │
│   • Encontros regulares (semanais ou quinzenais)                          │
│   • Grupo fechado de 4-8 pessoas                                         │
│   • Facilitador dedicado                                                 │
│   • Estrutura de confidencialidade                                       │
│                                                                             │
│   DIFERENCIAL:                                                            │
│   • Accountability mutuo                                                  │
│   • Aprendizado coletivo                                                 │
│   • Validacao de experiencias                                            │
│   • Custo compartilhado                                                  │
│                                                                             │
│   FUNCIONAL PARA:                                                          │
│   • Manter motivacao                                                     │
│   • Processar desafios diarios                                            │
│   • Desenvolvimento de comunidade                                       │
│   • Reduzir isolamento                                                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3. Comunidades de Pratica

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   COMUNIDADES DE PRÁTICA                                                  │
│                                                                             │
│   Características:                                                         │
│   • Grupo maior de pessoas em jornada similar                             │
│   • Encontros menos frequentes (mensais ou semestrais)                    │
│   • Baseada em praticas compartilhadas                                   │
│   • Sentido de pertencimento continuo                                    │
│                                                                             │
│   FORMADOS POR:                                                           │
│   • Meditacao ou praticas contemplativas                                 │
│   • Movimento corporal (yoga, tai chi)                                   │
│   • Arte ou expressao criativa                                           │
│   • Service ou servico comunitaria                                       │
│   • Tradições especificas (Santo Daime, etc)                             │
│                                                                             │
│   FUNCIONAL PARA:                                                          │
│   • Sustentacao de longo prazo                                           │
│   • Conexao espiritualcontinua                                          │
│   • Sentido de proposito                                                │
│   •apoio em momentos de crise                                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 4. Plataformas de Suporte

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   PLATAFORMAS DE SUPORTE                                                 │
│                                                                             │
│   Digitais:                                                               │
│   • Apps de meditacao (Insight Timer, Waking Up)                        │
│   • Grupos online de suporte                                             │
│   • Recursos de auto-ajuda                                               │
│   • Journaling digital                                                   │
│                                                                             │
│   Presenciais:                                                            │
│   • Centros de integracao                                               │
│   • Clinicas de acompanhamento                                          │
│   • Consultorias especializadas                                          │
│                                                                             │
│   ATENCAO:                                                                │
│   • Online não substitui acompanhamento profissional                     │
│   • Verificar credenciais de facilitadores                               │
│   • Cuidado com comunidades sem suporte qualificado                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Modelo de Aftercare em 3 Fases

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   AFTERCARE EM 3 FASES                                                    │
│                                                                             │
│   ══════════════════════════════════════════════════════════════════════     │
│                                                                             │
│   FASE 1: ESTABILIZACAO (Semanas 1-4)                                      │
│   ════════════════════════════════════════════════════                     │
│   • Acompanhamento intensivo                                              │
│   • Rotinas basicas de integracao                                        │
│   • Processamento inicial                                                │
│   • Estabilizacao emocional                                             │
│                                                                             │
│   FASE 2: INTEGRACAO ATIVA (Meses 2-6)                                     │
│   ════════════════════════════════════════════════════                     │
│   • Sessoes regulares (semanal ou quinzenal)                              │
│   • Desenvolvimento de praticas                                         │
│   • Teste de mudancas em vida real                                       │
│   • Ajustes conforme necessidade                                         │
│                                                                             │
│   FASE 3: SUSTENTACAO (Mes 7+)                                            │
│   ════════════════════════════════════════════════════                     │
│   • Check-ins menos frequentes                                          │
│   • Autonomia com suporte quando necessario                              │
│   • Participacao em comunidade                                          │
│   • Praticas continuadasindependentes                                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Aftercare para Diferentes Contextos

### Apos Ceremonia Isolada

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   APOS CERIMONIA ISOLADA                                                 │
│                                                                             │
│   MINIMO NECESSARIO:                                                      │
│   • 1-2 sessoes de integracao nas primeiras 2 semanas                    │
│   • Buddy de integracao (amigo de confianca)                            │
│   • Grupo de suporte ou circulo de integracao                            │
│   • Praticas diarias iniciadas                                          │
│                                                                             │
│   IDEAL:                                                                 │
│   • Acompanhamento terapeutico continuado                                │
│   • Participacao em grupo de integracao                                  │
│   • Retiros de aprofundamento                                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Apos Programa de Tratamento

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   APOS PROGRAMA DE TRATAMENTO                                            │
│                                                                             │
│   ESTRUTURA RECOMENDADA:                                                 │
│   • Acompanhamento intensivo (semanal) por 3-6 meses                     │
│   • Grupo de alumnus (pessoas que concluram programa)                    │
│   • check-ins mensais por 1-2 anos                                       │
│   • Acesso a suporte em crise                                           │
│   • Retiros trimestrais ou semestrais                                   │
│   • Plan de relapse prevention                                          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Custo-Beneficio de Aftercare

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   INVESTIMENTO EM AFTERCARE                                                │
│                                                                             │
│   CUSTOS:                                                                 │

│   • Temporais (horas de praticas, sessoes)                               │
│   • Emocionais (processamento, vulnerabilidade)                           │

│   • Reducao de risco de regresao                                         │
│   • Desenvolvimento de autonomia                                         │
│   • Crescimento continuo                                                │
│   • Communidade de suporte                                               │
│                                                                             │
│   CALCULO:                                                               │
│   Aftercare e investimento, nao gasto.                                    │
│   Os custos de nao fazer superam os custos de fazer.                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
