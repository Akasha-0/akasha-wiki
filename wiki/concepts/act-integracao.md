# Integracao Baseada em ACT

## Resumo

ACT (Acceptance and Commitment Therapy / Terapia de Aceitacao e Compromisso) oferece um framework comportamental poderoso para integrar experiencias psicodelicas. Os seis processos core do ACT fornecem ferramentas praticas para traduzir insights em mudancas duradouras.

## Origem do ACT

Desenvolvido por Steven Hayes na decada de 1980, ACT (Aceitacao e Compromisso) e uma abordagem da terapia cognitivo-comportamental contextual que combina aceitacao mindfulness com estrategias comportamentais para aumentar flexibilidade psicologica.

## Aplicacao na Integracao de Ayahuasca

### Passo 1: Aceitacao da Experiencia

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   ACEITACAO DA EXPERIENCIA                                                │
│                                                                             │
│   ══════════════════════════════════════════════════════════════════════     │
│                                                                             │
│   Abrir-se a experiencia completa (mesmo quando dificil)                  │
│                                                                             │
│   NA PRATICA:                                                              │
│   • Validar a experiencia como real e significativa                        │
│   • Permitir que surja o que precisa surgir                                │
│   • Não fugir ou suprimir insights                                        │
│   • Estar com o que é, mesmo quando desconfortavel                       │
│                                                                             │
│   EXERCICIO:                                                              │
│   "Esta experiencia e real. Ela me mudou de alguma forma.                │
│    Posso estar com o que surgiu, sem tentar muda-lo."                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Passo 2: Defusao Cognitiva

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   DEFUSAO COGNITIVA                                                       │
│                                                                             │
│   ══════════════════════════════════════════════════════════════════════     │
│                                                                             │
│   Observar pensamentos sem se identificar com eles                          │
│                                                                             │
│   "Eu tenho um pensamento" vs "Eu sou meu pensamento"                      │
│                                                                             │
│   A ayahuasca facilita isso atraves da dissolucao do ego                    │
│   e reducao da identificacao habitual com a mente                         │
│                                                                             │
│   NA PRATICA:                                                              │
│   • Notar quando a mente cria historias                                   │
│   • Perguntar: "Isso e um pensamento ou um fato?"                         │
│   • Criar espaco entre voce e seus pensamentos                            │
│   • Observar padroes sem julga-los                                        │
│                                                                             │
│   EXERCICIO:                                                              │
│   Quando perceber um pensamento como "Eu sou um fracassado":              │
│   Mude para "Estou tendo o pensamento de que sou um fracassado"           │
│   Note a diferenca em como isso se sente                                  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Passo 3: Acoes Valoriais

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   ACOES VALORIAS                                                          │
│                                                                             │
│   ══════════════════════════════════════════════════════════════════════     │
│                                                                             │
│   Identificar valores centrais descobertos na experiencia                  │
│   e criar compromissos de acao alinhados com eles                         │
│                                                                             │
│   NA PRATICA:                                                              │
│   • Identificar valores que emergiram                                     │
│   • Clarificar o que importa                                              │
│   • Definir passos concretos e mensuraveis                                │
│   • Agir apesar de resistencia                                            │
│                                                                             │
│   EXERCICIO:                                                              │
│   "Experiencia: percebi que passo tempo demais trabalhando                │
│    e pouco tempo com minha familia"                                      │
│                                                                             │
│   VALOR: Familia como prioridade central                                  │
│   AÇÃO: Agendar 2 noites/semana para familia                              │
│        Planejar vacaciones anuais                                         │
│        Criar ritual de desconexao do trabalho                             │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## ACT e Neuroplasticidade

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   ACT APOIA A INTEGRACAO NEUROLOGICA                                      │
│                                                                             │
│   ══════════════════════════════════════════════════════════════════════     │
│                                                                             │
│   DEFUSAO + ACOES VALORIAS:                                              │
│   • Cria novas vias neurais atraves de repeticao comportamental           │
│   • Reforca conexoes entre cortex e regioes de controle                   │
│   • Diminui forca de conexoes Automatic associated with old patterns      │
│                                                                             │
│   ACEITACAO:                                                              │
│   • Reduz respostas de estresse que interferem na plasticidade             │
│   • Cria estado neurologico favoravel a mudanca                           │
│                                                                             │
│   VALORES:                                                                │
│   • Fornece direcao e motivacao intrinsic                                 │
│   • Estabelece targets para consolidacao neural                           │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

```
│   │                                                                       │    │
│   │   DEFUNDIR:                                                           │    │
│   │   "Estou tendo o pensamento de que minha ansiedade                   │    │

│   │                                                                       │    │
│   │   VALORES:                                                            │    │

│   │   AÇÃO:                                                               │    │
```

## Comparacao: ACT vs Outros Modelos

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   COMPARATIVO: ACT vs OUTROS FRAMEWORKS                                   │

│   ───────────────────────────────────────────────────────────────────────     │

│   • Ha mudancas comportamentais especificas a fazer                        │
│   • Valores estao claros mas ha dificuldade de acting on them            │
│   • Ha padroes de evitamento a serem trabalhados                         │
│   • Raciocinio pratico e preferivel sobre exploracao experiencial        │
```
