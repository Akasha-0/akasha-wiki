# Problema Strip Mining

## Resumo

"Strip mining" e o fenomeno pelo qual experiencia psicodelicas profundas tem seus beneficios extraidos sem serem adequadamente processados, levando a perda gradual dos insights e transformacoes obtidos. Sem integracao, ate 60% dos beneficios podem ser perdidos.

## O que e Strip Mining

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   STRIP MINING: A METAFORA                                                 │
│                                                                             │
│   ══════════════════════════════════════════════════════════════════════     │
│                                                                             │
│   Mineração a ceu aberto remove camadas de solo para extrair               │
│   minerio valioso. Apos a extracao, o terreno fica devastado -            │
│   sem recursos, sem vida, sem estrutura.                                   │
│                                                                             │
│   DA MESMA FORMA:                                                         │
│   Experiencias psicodelicas contêm insights valiosos (minerio).           │
│   Quando mineramos sem integrar, extraimos o valor mas devastamos         │
│   o potencial de transformacao.                                            │
│                                                                             │
│   EXPERIENCIA ↓ ↓ ↓                                                        │
│   Extractamos insights → Sem processar → Terreno devastado               │
│   (sem mudanca duradoura)                                                 │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

│   ══════════════════════════════════════════════════════════════════════     │

│   □ Defina intencoes claras                                               │
│   □ Tenha um buddy de integracao confirmado                               │
│   □ Programe tempo protegido pos-experiencia (1-2 semanas)                │
│   □ Identifique praticas diarias que vai manter                          │
│   □ Tenha contato de facilitador/terapeuta para follow-up                  │
│   □ Programe sessoes de integracao                                       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## O que Fazer Se Ja Ocorreu

### Recovery

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   RECUPERACAO DO STRIP MINING                                              │
│                                                                             │
│   Se voce reconhece que extraiu beneficios sem integrar:                   │
│                                                                             │
│   1. ACEITE                                                              │
│      • Nao julgue a si mesmo                                              │
│      • Reconheca que isso e comum                                        │
│      • A conscientizacao e o primeiro passo                             │
│                                                                             │
│   2. REVISE                                                              │
│      • Volte ao que lembrar da experiencia                                │
│      • Faca journaling do que ainda esta presente                        │
│      • Reconecte-se com intencionalidade original                        │
│                                                                             │
│   3. INTEGRE AGORA                                                        │
│      • Ainda ha tempo - a janela de plasticidade pode ter fechado         │
│        parcialmente, mas praticas ainda funcionam                        │
│      • Comece um processo formal de integracao                           │
│      • Busque suporte profissional se necessario                         │
│                                                                             │
│   4. PLANEJE O FUTURO                                                    │
│      • Proximas experiencias devem incluir integracao                    │
│      • Considere retreat de integracao                                   │
│      • Crie estruturas de suporte continuas                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
