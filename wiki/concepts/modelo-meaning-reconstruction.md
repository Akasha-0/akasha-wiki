# Meaning Reconstruction (A-MR)

## Resumo

O modelo A-MR (Ayahuasca + Meaning Reconstruction) combina ayahuasca com terapia de Reconstrução de Significado para tratamento de luto complicado. O ensaio clínico publicado no Nature Scientific Reports (2025) demonstrou superioridade sobre terapia isolada.

## Origem

A Reconstrução de Significado é uma abordagem psicoterapêutica desenvolvida para ajudar pessoas a encontrar sentido após perdas significativas. O modelo A-MR integra esta abordagem com ayahuasca.

## Ensaio Clínico Nature (2025)

### Metodologia
- **Design:** 3 braços, alocação sequencial (open-label)
- **Participantes:** 84 pessoas com luto prolongado
- **Intervenção:** Ayahuasca + Meaning Reconstruction

### Resultados
- **A-MR superou terapia isolada** significativamente
- Efeitos mantidos por **3 meses** após intervenção
- Primeiro ensaio clínico com ayahuasca para luto complicado
- Primeiro ensaio a usar ayahuasca para reconstrução de significado

## Mecanismo Proposto

```
Luto complicado → Memória da perda codificada rigidamente
        ↓
Ayahuasca:
1. Reduz DMN → menor rigidez da narrativa
2. Acessa memória com distância emocional
3. Integra experiência com novo significado
4. Terapia MR → ferramentas para processar a perda
        ↓
Resultado: Capacidade de continuar vivendo com a perda
```
