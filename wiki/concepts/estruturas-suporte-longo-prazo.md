# Estruturas de Suporte para Integração a Longo Prazo

## Resumo

A integração é um processo contínuo, não um evento único. Este documento apresenta estruturas e práticas de suporte para manter e aprofundar os benefícios de experiências com ayahuasca ao longo do tempo.

```
┌─────────────────────────────────────────────────────────────────────────────┐

│   • Útil para processamento profundo de experiências                     │
│                                                                             │
│   PSICÓLOGO CLÍNICO:                                                     │
│   • Útil para síntomas persistentes (ansiedade, depression)                │
│   • Abordagens evidence-based (CBT, EMDR, etc.)                          │
│   • Quando há indicação clínica                                         │
│                                                                             │
│   PSICOTERAPEUTA ESPIRITUAL:                                             │
│   • Foco em desenvolvimento espiritual e transpessoal                   │
│   • Útil para búsqueda de significado e propósito                        │
│   • Integração de experiencias místicas                                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

**O que significa:** Criar espaço interno e externo para que a experiência seja fully processed.

## O Problema da "Strip Mining": Por Que Benefícios Se Perdem

### O Que É "Strip Mining"

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   STRIP MINING: DEFINIÇÃO                                                  │
│                                                                             │
│   ══════════════════════════════════════════════════════════════════════    │
│                                                                             │
│  Strip mining (mineração rasgada) ocorre quando benefícios de uma         │

│   integração, levando à perda dos beneficios ao longo do tempo.           │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
