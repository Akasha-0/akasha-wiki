# Circulos de Integracao

## Resumo

Circulos de integracao sao grupos de pessoas que se encontram regularmente para compartilhar experiencias, processar insights e supporting mutuamente em suas jornadas com ayahuasca e outras praticas de.expandacao de consciencia. Oferecem validacao, perspectiva e accountability que complementam o trabalho individual.

## Tipos de Circulos

### Círculo de Compartilhamento

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   CÍRCULO DE COMPARTILHAMENTO                                             │
│                                                                             │
│   Características:                                                         │
│   • Estrutura circular, iguais ao redor                                   │
│   • Palavra livre com respeito                                           │
│   • Facilitador que contém                                                │
│   • Escuta sem julgamento                                                │
│   • Mais tradicional, ancestral                                           │
│                                                                             │
│   IDEAL PARA:                                                              │
│   • Processamento inicial de experiencias                                │
│   • Validacao e normalizacao                                             │
│   • Conexao comunitaria                                                  │
│   • Quando há material emocional a compartilhar                          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Círculo de Processamento

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   CÍRCULO DE PROCESSAMENTO                                                │
│                                                                             │
│   Características:                                                         │
│   • Estrutura mais terapêutica                                           │
│   • Foco em insight pessoal                                              │
│   • Facilitador ativo                                                    │
│   • Exercícios específicos                                              │
│   • Mais clínico                                                         │
│                                                                             │
│   IDEAL PARA:                                                              │
│   • Trabalho mais profundo                                               │
│   • Identificacao de padroes                                             │
│   • Processamento de material complexo                                   │
│   • Quando ha necessidade de investigacao                               │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Estrutura de uma Sessao

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   ESTRUTURA SUGERIDA                                                      │
│                                                                             │
│   ══════════════════════════════════════════════════════════════════════     │
│                                                                             │
│   DURACAO: 90-120 minutos                                                 │
│   TAMANHO: 4-8 pessoas                                                    │
│   FREQUENCIA: Semanal ou quinzenal                                       │
│                                                                             │
│   ESTRUTURA:                                                              │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │ ABERTURA (10-15 min)                                               │  │
│   │ • Chegada e centramento                                            │  │
│   │ • Intencao do encontro                                             │  │
│   │ • Orientações básicas                                             │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │ CHECK-IN (15 min)                                                   │  │
│   │ • Como voce esta desde nosso ultimo encontro?                     │  │
│   │ • Em uma palavra: como voce esta?                                 │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │ TEMA OU PROMPT (30 min)                                            │  │
│   │ • "Como estou vivendo meus insights no dia a dia?"                 │  │
│   │ • "Que padrao eu percebi em mim recentemento?"                     │  │
│   │ • "Onde estou sendo consistente/inconsistente?"                   │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │ COMPARTILHAMENTO LIVRE (30-40 min)                                 │  │
│   │ • Cada pessoa compartilha por 5-7 minutos                         │  │
│   │ • Os outros escutam sem interrupcao                                │  │
│   │ • Sem conselhos nao solicitados                                     │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │ REFLEXÃO COLETIVA (15-20 min)                                      │  │
│   │ • Temas comuns identificados                                      │  │
│   │ • Insights emergentes                                             │  │
│   │ • Padrões observados                                               │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│   ┌─────────────────────────────────────────────────────────────────────┐  │
│   │ FECHAMENTO (10 min)                                                 │  │
│   │ • Síntese do encontro                                              │  │
│   │ • Compromissos                                                     │  │
│   │ • Próximo encontro                                                 │  │
│   └─────────────────────────────────────────────────────────────────────┘  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Regras Fundamentais

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   REGRAS DOS CÍRCULOS                                                     │
│                                                                             │
│   ══════════════════════════════════════════════════════════════════════     │
│                                                                             │
│   1. CONFIDENCIALIDADE                                                    │
│      • "O que é dito aqui, fica aqui"                                     │
│      • Protecao do espaco e das historias                               │
│      • Confianca mútua                                                    │
│                                                                             │
│   2. SEM JULGAMENTO                                                      │
│      • Nao interpretar ou analisar                                       │
│      • Escuta plena e presença                                           │
│      • Aceitacao incondicional                                           │
│                                                                             │
│   3. SEM CONSELHOS NAO SOLICITADOS                                       │
│      • Nao oferecer solucoes a menos que solicitado                      │
│      • Focar em presencia, nao em correcao                               │
│      • Se precisar advice, perguntar: "Posso te pedir uma opiniao?"      │
│                                                                             │
│   4. PRESENCE                                                           │
│      • Celular no silencioso                                             │
│      • Chegar no horario                                                 │
│      • Permanecer ate o fim                                              │
│                                                                             │
│   5. RESPONSABILIDADE PESSOAL                                           │
│      • Cuidar de si mesmo primeiro                                       │
│      • Compartilhar apenas o que se sente seguro                         │
│      • Respeitar limites proprios e alheios                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Como Encontrar ou Criar um Círculo

### Opções Disponíveis

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   ENCONTRAR CÍRCULOS                                                      │
│                                                                             │
│   1. COMUNIDADES EXISTENTES                                             │
│      • Grupos de meditacao                                               │
│      • Comunidades de praticas holisticas                               │
│      • Redes de pessoas em jornada psicodelica                         │
│                                                                             │
│   2. TERAPEUTAS E FACILITADORES                                          │
│      • Pergunte a terapeutas e facilitadores                            │
│      • Eles frequentemente conhecem ou facilitam grupos                 │
│                                                                             │
│   3. RETIROS E EVENTOS                                                   │
│      • Retiros de integracao                                             │
│      • Eventos de empresas como ICEERS, Chacruna                         │
│      • Conferencias como Psychedelic Science                            │
│                                                                             │
│   4. ONLINE                                                              │
│      • Grupos virtuais (com cuidado)                                      │
│      • Communities of practice online                                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Criar Seu Proprio Círculo

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   CRIAR UM CÍRCULO                                                       │
│                                                                             │
│   PASSOS:                                                                  │
│   1. Reúna 3-6 pessoas interessadas em integracao                       │
│   2. Defina intencao e formato do grupo                                 │
│   3. Estabeleca regras e acordos basicos                              │
│   4. Escolham轮流 facilitacao ou contratem profissional                   │
│   5. Comecem com encontros semanais por 4-6 semanas                    │
│   6. Avaliem e ajustem conforme necessidade                             │
│                                                                             │
│   DICAS:                                                                  │
│   • Comecem devagar, sem pressa                                          │
│   • Priorizem constincia sobre intensidade                              │
│   • Cuidem da confidencialidade desde o inicio                           │
│   • Se nao ha facilitador experiente, leiam bastante antes              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Integracao Online vs Presencial

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   COMPARATIVO: ONLINE VS PRESENCIAL                                       │
│                                                                             │
│   ONLINE:                                                                 │
│   • Acesso facilitado                                                    │
│   • Pode participar de qualquer lugar                                    │
│   • Menos intimacy (ausencia de linguagem corporal plena)                 │
│   • Funcional para check-ins basicos                                     │
│                                                                             │
│   PRESENCIAL:                                                             │
│   • Presenca completa                                                   │
│   • Linguagem corporal esta presente                                    │
│   • Energia compartilhada                                              │
│   • Mais indicado para trabalho profundo                                 │
│                                                                             │
│   RECOMENDACAO:                                                          │
│   • Para trabalho superficial: online funciona bem                       │
│   • Para processamentos profundos: presencial e preferivel               │
│   • Ideal: combinacao de ambos                                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
