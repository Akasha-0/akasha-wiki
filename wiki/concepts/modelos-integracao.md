## 1. Modelo THRIVE (6 Dimensões)

### Origem
Desenvolvido por terapeutas psicodélicos, o THRIVE oferece um framework holístico para integração.

### As Dimensões

| Letra | Dimensão | Foco | Práticas |
|-------|----------|------|----------|
| T | Trust (Confiança) | Trusted Space | Estabelecer segurança |
| H | Holistic (Holístico) | Saúde completa | Corpo, mente, espírito |
| R | Relational (Relacional) | Conexões | Vínculos, comunidade |
| I | Internal (Interno) | Self | Insight, autoconsciência |
| V | Values (Valores) | Propósito | Alinhamento valorial |
| E | Existential (Existencial) | Transcendência | Significado, propósito |

### Alocação de Energia (Modelo 60-30-10)
```
60% → INTERNAL SELF (Psicoterapia individual, journaling, meditação)
30% → RELATIONAL (Grupos, terapia de casal, comunidade)
10% → EXISTENTIAL (Exploração de propósito, serviço)
```

### Aplicação na Integração de Ayahuasca

#### Passo 1: Aceitação da Experiência
- Abrir-se à experiência completa (mesmo quando difícil)
- Não fugir ou suprimir insights
- Validar a experiência como real e significativa

#### Passo 2: Defusão Cognitiva
- Observar pensamentos sem se identificar com eles
- "Eu tenho um pensamento" vs "Eu sou meu pensamento"
- A ayahuasca facilita isso através da dissolução do ego

#### Passo 3: Ações Valoriais
- Identificar valores centrais descobertos na experiência
- Criar compromissos de ação alinhados com esses valores
- Definir passos concretos e mensuráveis

### Exemplo Prático

```
Experiência: "Eu percebi que passo tempo demais trabalhando e pouco 
              tempo com minha família"

ACT Aplicado:
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   ACEITAÇÃO: Validar o insight sem culpa ou julgamento                     │
│   DEFUSÃO: "Eu sou alguém que escolheu trabalho" vs "Eu sou meu trabalho" │
│   VALORES: Família como prioridade central                                 │
│   AÇÃO: Agendar 2 noites/semana para família, férias planejadas          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 8. Psicoterapia Transpessoal (Stanislav Grof)

### Origem
Desenvolvida por Stanislav Grof como a "quarta força" da psicologia, a transpessoal integra dimensões espirituais e transcendentes no processo terapêutico. Grof mapeou territórios da psyche além do ego, identificando que estados não-ordinários não são patológicos — são oportunidades de cura.

### Dimensões da Consciência

| Dimensão | Descrição | Fenômenos Típicos |
|----------|-----------|-------------------|
| **Hylotrópica** | Consciência ordinária | Ego intacto, tempo linear, realidade consensual |
| **Holotrópica** | Estados expandidos | Transcendência do ego, tempo não-linear, unidade |

### Matrizes Perinatais (BPM)

Grof identificou 4 matrizes inconscientes relacionadas ao nascimento que moldam padrões psicopatológicos:

```
MATRIZES PERINATAIS (Grof)
═══════════════════════════

BPM I: PRÉ-UNIVOCAL (Mergulho no Absoluto)
• União biológica com a mãe
• Vivência de beatitude, oceanicidade

BPM II: BIOLÓGICO (Conflito e Separação)
• Rompimento da união
• Experiências de terror, caos, ameaça de morte

BPM III: DINÂMICO (Luta pela Sobrevivência)
• Combinação de tendências reprodutivas e perigo
• Experimentação de estados alterados intensos

BPM IV: SEPARAÇÃO (Nascimento e Renascimento)
• Experiência de renascimento
• Estados de transcendência e renovação
```

### Sistemas COEX (Systems of Concentrated Experience)

Grof propôs que memórias de diferentes vidas ou experiências transpessoais podem agrupar-se em "sistemas COEX" — clusters emocionais que influenciam padrões atuais:

```
SISTEMA COEX
═════════════

┌────────────────────────────────┐
│      TEMA CENTRAL              │
│  (ex: "Sou indigno de amor")    │
│         ▲                      │
│    ┌────┴────┐                 │
│    │         │                 │
│  Memória   Memória             │
│  atual    transpessoal         │
│  (infância) (outra vida)       │
│    │         │                 │
│    └────┬────┘                 │
│         ▼                      │
│  PADRÃO COMUM DE RESPOSTA       │
│  (evitação de intimidade)       │
└────────────────────────────────┘
```

### Aplicação na Integração de Ayahuasca

A ayahuasca abre portais para material transpessoal. A integração transpessoal ajuda a:
- Contextualizar experiências além do ego
- Trabalhar com sistemas COEX identificados
- Processar matrizes perinatais ativadas
- Integrar insights de dimensões espirituais

## 9. Somatic Experiencing (Peter Levine)

### Origem
Somatic Experiencing (SE), desenvolvido por Peter Levine, oferece ferramentas para processar trauma que a ayahuasca pode ter acessado mas não resolvido completamente.

### Modelo SIBAM

O SE trabalha com o ciclo completo de processamento somático:

```
MODELO SIBAM (Somatic Experiencing)
═══════════════════════════════════

    ┌─────────┐
    │ SENSAÇÃO │  ← Primeiro nível (corpo)
    │   (S)   │    Como os pés tocam o chão?
    └────┬────┘    Como é a respiração agora?
         │
         ▼
    ┌─────────┐
    │ IMAGEM  │  ← Transformação sensorial
    │   (I)   │    Imagens que surgem?
    └────┬────┘
         │
         ▼
    ┌───────────┐
    │ COMPORTAMENTO│← Respostas automáticas
    │    (B)    │    Quer fugir? Ficar? Lutar?
    └─────┬─────┘
         │
         ▼
    ┌─────────┐
    │ AFETO   │  ← Emoções emergem
    │   (A)   │    Chorar? Raiva? Medo?
    └────┬────┘
         │
         ▼
    ┌─────────┐
    │SIGNIFICADO│← Construção de narrativa
    │   (M)   │    O que isso significa sobre mim?
    └─────────┘

OBS: O trabalho com ayahuasca frequentemente
para em S-I-B mas não chega ao A-M sem integração.
O SE ajuda a completar o ciclo.
```

### Por Que SE Complementa Ayahuasca

- Ayahuasca acessa conteúdo somático profundo
- SE fornece ferramentas para processar o corpo
- Juntos: acesso + processamento = integração completa

## 10. EMDR (Eye Movement Desensitization and Reprocessing)

### Origem
Desenvolvido por Francine Shapiro, o EMDR utiliza estimulação bilateral para facilitar o reprocessamento de memórias traumáticas.

### Mecanismo

```
REPROCESSAMENTO COM EMDR
═════════════════════════

    Memória traumática mal processada
              │
              ▼
    ┌─────────────────────────┐
    │  CANAL DE RESTING       │  ← Acesso à memória
    │  (componente: exposição)│    através de BLS
    └─────────────────────────┘
              │
              ▼
    Dessensibilização + Reconsolidação
              │
              ▼
    Nova rede neural com significado
    integrado (memória adaptativa)
```

### Nota sobre a Estimulação Bilateral

Pesquisas indicam que o componente essencial é a exposição e reconsolidação — não a estimulação bilateral em si. Isso é relevante para integração psicodélica, onde a "exposição" ocorre durante a experiência com ayahuasca.

### Aplicação na Integração

- Reprocessar memórias traumáticas acessadas durante cerimônia
- Dessensibilizar respostas emocionais intensas
- Reconsolidar com novo significado

1. **Reimaginar**: Novas perspectivas sobre velhos problemas
2. **Recontextualizar**: O mesmo evento, novo significado
3. **Reintegração**: Unificar aspectos escindidos do self
