# Set and Setting

## Resumo

Set (mentalidade) e Setting (ambiente) são os dois fatores externos mais importantes para determinar a segurança e eficácia de experiências com ayahuasca. Diretrizes ReSPCT foram publicadas em 2025 para padronizar seu reporte em estudos clínicos.

## Definições

### Set (Mentalidade)
- Estado mental e emocional do participante
- Intenção e preparação
- Experiências anteriores
- Expectativas
- Barreiras psicológicas

### Setting (Ambiente)
- Ambiente físico (espaço, iluminação, som)
- Ambiente social (companhia, presença de facilitadores)
- Ambiente temporal (hora do dia, duração)
- Componentes objetivos e subjetivos

## Diretrizes ReSPCT (2025)

As diretrizes ReSPCT (Recommendations for Safety in Psilocybin and Psychedelic Research and Clinical Translation) estabelecem:

### Componentes do Setting
1. **Físico:** Espaço seguro, confortável, privativo
2. **Social:** Presença de guides/cuidadores treinados
3. **Temporal:** Tempo adequado sem interrupções
4. **Objetivo:** Aspectos mensuráveis do ambiente
5. **Subjetivo:** Percepção individual do ambiente

## Ciclo Completa de Preparação

```
1. PREPARAÇÃO (Set) → Meditação, intenção, reflexão
2. CERIMÔNIA → Experiência com ayahuasca
3. PÓS-CERIMÔNIA → Integração, processamento
4. PRÁTICA CONTÍNUA → Manutenção, crescimento
5. IDENTIDADE TRANSFORMADA → Novo Self
```

## Importância

O set e setting adequados:
- Reduzem risco de experiências difíceis (bad trips)
- Maximizam potencial terapêutico
- Facilitam integração posterior
- Promovem segurança física e psicológica
