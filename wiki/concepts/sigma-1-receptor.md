# Sigma-1 Receptor

O sigma-1 receptor é uma chaperona transmembrana encontrada em todo o sistema nervoso central e tecidos periféricos. DMT é seu **ligante endógeno**, o que confere propriedades neuroprotetoras e anti-inflamatórias únicas. Este receptor é fundamentalmente diferente dos clássicos receptores de neurotransmissores - atua como um modulador de chaperona que regula a homeostase celular em resposta ao estresse.

## Descoberta e Características

### Fontanilla et al. (2009) - A Descoberta Fundamental

O artigo de Fontanilla et al. published in *Proceedings of the National Academy of Sciences* (PNAS) established DMT as the first endogenous ligand for sigma-1:

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   FONTANILLA 2009: DESCOBERTA HISTÓRICA                                   │
│                                                                             │
│   ══════════════════════════════════════════════════════════════════════    │
│                                                                             │
│   "DMT is an endogenous ligand for sigma-1 receptors"                     │
│   Fontanilla D et al. (2009) PNAS 106(50):21383-21388                    │
│                                                                             │
│   IMPLICAÇÃO: O cérebro produz sua própria "droga psicodélica"           │
│   e o sigma-1 é parte do sistema receptor para essa molécula.              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Características Únicas

- **Tipo**: Receptor órfão (não ligado a peptídeos opioides)
- **Estrutura**: Chaperona transmembrana de 223 aminoácidos
- **Localização**: Predominantemente na membrana do retículo endoplasmático (ER), especialmente em neurônios e células immunitárias
- **Expressão**: Encontrado em todo o SNC, включая hipocampo, córtex, astrócitos
- **Distinção**: Não é um receptor de neurotransmissor clássico - é um **modulador de chaperona**

### Mecanismo de Sinalização em 5 Etapas

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   SINALIZAÇÃO DMT → SIGMA-1 → NEUROPROTEÇÃO                               │
│                                                                             │
│   ══════════════════════════════════════════════════════════════════════    │
│                                                                             │
│   PASSO 1: LIGAÇÃO                                                         │
│   DMT liga-se ao sigma-1 na interface ER-membrana                         │
│                                                                             │
│   PASSO 2: DISSOCIAÇÃO DO COMPLEXO                                          │
│   DMT-sigma-1 se dissocia da proteína BIP ( chaperona do ER)              │
│   → sigma-1 fica em estado "ativo"                                         │
│                                                                             │
│   PASSO 3: MODULAÇÃO DE CANAIS IÔNICOS                                    │
│   O complexo ativo modula canais de Ca2+ (IP3R, canais de Ca²⁺)          │
│   → Liberação de Ca²⁺ do ER para citoplasma                               │
│                                                                             │
│   PASSO 4: ATIVAÇÃO DE VIAS                                                │
│   ↑ Ca²⁺ intracelular ativa:                                               │
│   • PI3K/Akt → sobrevivência celular                                       │
│   • MAPK/ERK → proliferação e diferenciação                              │
│   • BDNF expression → neuroplasticidade                                   │
│                                                                             │
│   PASSO 5: RESULTADO                                                       │
│   • Genes anti-apoptóticos expressos                                       │
│   • Fatores neurotróficos liberados                                        │
│   • Sobrevivência e diferenciação neuronal                                │
│   • Resposta anti-inflamatória                                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Funções Biológicas do Sigma-1

### 1. Neuroproteção

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   NEUROPROTEÇÃO VIA SIGMA-1                                               │
│                                                                             │
│   • Sobrevivência celular sob estresse                                      │
│   • Proteção contra estresse oxidativo                                       │
│   • Facilitação de diferenciação neuronal                                   │
│   • Manutenção de funções mitocondriais                                   │
│   • Redução de morte celular programada (apoptose)                        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 2. Regulação de Estresse do ER

O sigma-1 atua especificamente em regiões de **MAMs** (Mitochondria-Associated Membranes), que são zonas de contato entre ER e mitocôndria:

- Detecta condições de estresse no ER (acúmulo de proteínas mal dobradas)
- Modula transferência de Ca²⁺ para mitocôndria
- Regula metabolismo energético celular
- Ativa respostas de unfolded protein response (UPR)

### 3. Imunomodulação

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   MODULAÇÃO IMUNE VIA SIGMA-1                                              │
│                                                                             │
│   Redução de citocinas pró-inflamatórias:                                 │
│   • IL-1β, IL-6, TNF-α reduzidos                                         │
│                                                                             │
│   Aumento de citocinas anti-inflamatórias:                                │
│   • IL-10 aumentado                                                       │
│                                                                             │
│   Modulação de células imunes:                                            │
│   • Macrófagos, micróglia, células dendríticas                            │
│   • Favorece perfil anti-inflamatório M2                                  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

```
│                                                                             │
│   Tolerância clássica (redução de resposta com uso repetido)             │
│   parece ser MENOS pronunciada no sigma-1 comparada a outros receptores.   │
│                                                                             │
│   No entanto:                                                              │
│   • Uso crónico pode levar a downregulation do receptor                   │

```

### Regulação por Estado Fisiológico

O sigma-1 é único entre receptores porque:
- Sua atividade é regulada por condições de estresse celular
- Funciona como **sensor de estresse integrado**
- A ligação de DMT potencializa a capacidade de resposta ao estresse
- Não há simplesmente "ligado/desligado" - é um **modulador de ganho**

### Por Que o Sigma-1 É Diferente

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   SIGMA-1 ≠ RECEPTOR CLÁSSICO DE NEUROTRANSMISSOR                         │
│                                                                             │
│   Receptor clássico:                                                      │
│   • Liga neurotransmissor → abre/fecha canal ou ativa GPCR               │
│   • Resposta rápida (ms)                                                  │
│   • Desensibilização rápida                                               │
│                                                                             │
│   Sigma-1 como chaperona:                                                  │
│   • Liga DMT → modula estado de chaperona                                 │
│   • Resposta lenta (min-horas)                                            │
│   • Regula respostas ao estresse celular                                   │
│   • Não dessensibiliza da mesma forma                                     │
│                                                                             │
│   Implicação clínica:                                                     │
│   • Efeitos do DMT via sigma-1 são mais sustentados                        │
│   • Não há necessidade de uso frequente para manter efeito                │
│   • Integração com práticas (meditação) pode potencializar                │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
